# Base ETH/WETH Swap Bot

Bot Python untuk melakukan swap antara ETH dan WETH di Base Network.

## 🚀 Fitur
- Swap ETH → WETH (Wrap)
- Swap WETH → ETH (Unwrap) 
- Mode manual dan otomatis
- Support Base Mainnet
